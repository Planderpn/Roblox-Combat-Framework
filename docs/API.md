# API

## Attack

```lua
Combat:Attack(player)
```

Starts the next attack in the combo.

---

## Dash

```lua
Combat:Dash(player, direction)
```

Moves the player in the given direction.

Directions:
- Forward
- Back
- Left
- Right

---

## Blocking

```lua
Combat:SetBlocking(player, enabled)
```

Turns blocking on or off.

---

## Config

Most values can be edited inside `Config.lua`.
